## 📌 About The Project

IELTS AI Checker is a full-stack web application that analyzes IELTS Writing essays and provides:

* Grammar mistake detection
* Highlighted errors
* Corrected text suggestions
* Estimated IELTS band scores
* Detailed feedback for:

  * Task Response
  * Coherence & Cohesion
  * Lexical Resource
  * Grammar

This project uses a smart rule-based scoring system and free APIs instead of paid AI tools.

---

## ✨ Features

* ✅ Grammar checking using LanguageTool API
* ✅ Highlight incorrect words in the essay
* ✅ Show corrected text suggestions
* ✅ Generate IELTS band score estimate
* ✅ Separate score breakdown for:

  * Task Response
  * Coherence & Cohesion
  * Lexical Resource
  * Grammar
* ✅ Smart rule-based scoring system
* ✅ Responsive and clean UI
* ✅ Free deployment with Netlify and Render

---

## 🧠 How It Works

```text
User enters essay
        ↓
Frontend sends essay to backend
        ↓
Backend analyzes grammar, vocabulary, and structure
        ↓
Band score + detailed feedback generated
        ↓
Results shown on frontend
```
